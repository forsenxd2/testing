Tạo Template DOCX mẫu cho Bảng chào Blanca City

Nội dung template mẫu:

BẢNG CHÀO BLANCA CITY Chức vụ: {chuc_vu} Tên cán bộ: {ten_can_bo} Số điện thoại (Zalo): {so_dien_thoai} Dự án Blanca City - Nơi an cư lý tưởng cho gia đình bạn. Thông tin liên hệ: - Điện thoại: {so_dien_thoai} - Zalo: {so_dien_thoai} QR Code Zalo để liên hệ nhanh: {qr_code} Trân trọng, {ten_can_bo} {chuc_vu}

Hướng dẫn sử dụng:

  1. Nhấn nút "Tạo file template DOCX mẫu" để tải về file template
  2. Mở file bằng Microsoft Word
  3. Chỉnh sửa nội dung theo ý muốn (giữ nguyên các placeholder {chuc_vu}, {ten_can_bo}, {so_dien_thoai}, {qr_code})
  4. Lưu file với tên "bangchao.docx" vào thư mục "pdf/"
  5. Sử dụng trong trang bangchaoblanca.html
